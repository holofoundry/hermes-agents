# Skill: DCF Model

Use this skill when the user asks for intrinsic value, discounted cash flow analysis, WACC sensitivity, terminal value, enterprise value, or equity value.

Minimum inputs are company name or ticker, historical revenue, EBIT or EBITDA margin, tax rate, depreciation and amortization, capital expenditure, net working capital changes, cash, debt, shares outstanding, discount rate, terminal growth rate, and projection assumptions.

Build the analysis in this order: normalize historicals, project revenue, project operating profit, calculate NOPAT, add back D&A, subtract capex and working capital investment, discount free cash flow, calculate terminal value, bridge enterprise value to equity value, calculate per-share value, and run sensitivities.

Always show assumptions. Always flag missing inputs. Do not invent live market data. If data is estimated, label it as estimated.
